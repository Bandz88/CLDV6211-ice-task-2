# Kias Kar Kompany

## Overview
This is an ASP.NET Core MVC application that manages car records using full CRUD functionality.

## Features
- Add new cars
- View car list
- Edit car details
- Delete cars

## Technologies
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server

## Setup
1. Clone repo
2. Update connection string in appsettings.json
3. Run:
   dotnet ef database update
4. Run app:
   dotnet run
